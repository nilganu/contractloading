================================================================
VALUE FORMATTING RULES
================================================================

- Dates: ISO "YYYY-MM-DD".
- Days: weekday mask "1234567" (1=Mon..7=Sun); all-week = "1234567";

- Country Code: ISO 3166-1 alpha-2, uppercase (eg "GR", "MV", "EG").
- Currency / Customer Price Currency: 3-letter code (EUR, USD, ...).
- Status: "Open" (default) or "On Request".
- Check-In / Check-Out: "HH:MM" 24h (eg "14:00", "12:00") if known.
- Star Rating: one of "1 Star".."7 Star", "Boutique Hotel", "Self Catering".
- Bed Type: King / Queen / Twin / Double / Single / Sofa bed / ... or null.
- Meal Plan: "Bed and Breakfast", "Half board", "Full board", "All
  inclusive", "Room only", "Ultra All Inclusive", ... (use the contract's
  wording if a recognisable equivalent).
- Numbers as JSON numbers (not strings); unknown -> null.
- "Free" / "FOC" / "included" for a child rate -> Price 0.
- "On Request" / "N/A" -> null (not 0).
